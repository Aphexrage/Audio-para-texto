<h1 align="center">Classificador de sons ambientes - PROT</h1>

Este projeto consiste no desenvolvimento de um **sistema inteligente de transcrição de áudio**, com foco na **acessibilidade para pessoas com deficiência auditiva**.

O algoritmo identifica dois tipos principais de som no ambiente:

- 🔊 **Barulhos gerais**
- 🗣️ **Vozes humanas**

Ao detectar barulhos gereais:

  - Mostra ao usuario o nome do som.

Ao detectar a fala de uma pessoa, o sistema:

1. Grava automaticamente o áudio.
2. Transcreve o conteúdo falado.
3. Atribui a fala ao nome da pessoa (caso previamente identificada).

---

### 🧠 Como Funciona

Sempre que alguém fala, a transcrição será exibida de forma clara e identificada, por exemplo:

```txt
João: Podemos começar a reunião agora.
```
